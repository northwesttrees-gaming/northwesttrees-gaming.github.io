POSTED: OCT 06th, 2021

It's been a while lol. Not exactly what I was hoping for, I have been working on a few other mods in the time since the last post. I made a lot of progress however the remote workspace bumbed up in MCreator and I lost all the work I did sadly. The good news is the GitHub files still worked but it was from 0.2.1 so I have to add back things slowly now. The update would have easily taken me more months to compley and the wiki maybe even double that so it may have been for the better.

I still have the resources saved on my external hard-drive so I can add the features planned in a future update maybe a little bit at a time but it will not be in a huge major update like I was planning. The good news is I did some minor work on improving the Auto Farming Machine fixing the tags for it and some other elements making them a lot better, and even found some time to add a new feature for upgrades, some GUI improvements and such, the other blocks will have a overhaul soon too. But this should be a good start to get me back into making the mod.

This should take over the weekend to do but will be worth it in the end as from testing its much more enjoyable with the new sounds.

ITEMS COMPLETE 0.3.0

POSTED: AUG 18th, 2021

Currently progress is going well, many things have moved to the testing phase, and all the items are now added and done. Blocks are not to far to follow. After which I will be focusing on getting the reworking done for the block already added in the last two update. May recipes are still needing testing, I think I may have tested them but I don't remember doing it so I will be testing them again just to be safe.
