Wiki - Machine Wrench

Overview

The Machine Wrench is used for linking devices to all kinds of machines to expand thier uses. How Machine Wrenches work is you right-click on the block you want to connect the machine to, then click on the machine with the same item. When right-clicking on a device NBT variables are stored to the Machine wrench that gives the machine important properties for it to sync to the device. See below for more on NBT Variables.

NBT

Position

The position variables is a number variable, and they are used to store the device's location.

  • SavedPosX

    This saves the X location of the device.

  • SavedPosY

    This saves the Y location of the device.

  • SavedPosZ

    This saves the Z location of the device.

Source Block

The source block variable is a string variable, and it is used to keep track of what device is clicked on.

  • SavedSourceBlock
    • BreakerPanel

      This is the value for the Breaker Panel device.

    • Padlock

      This is the value for the Padlock device.

    • RemoteItemTeleporter

      This is the value for the Remote Item Teleporter device.

Has Save Block

The has saved block variable is a logic variable, and it is used to keep track if the item can be stored or not.

  • SavedSourceBlock
    • true

      When true the item can pass the variables to a machine.

    • false

      When false the item can collect variables from a device.
