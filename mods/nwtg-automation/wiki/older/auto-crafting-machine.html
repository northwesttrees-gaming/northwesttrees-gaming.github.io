Overview

The Auto Crafting Machine is used to craft things automatically. Not all items are currently supported for auto crafting, each update we release more recipes though.

Functions

The block needs two chests on the right and left of the block for the direction its facing. The chest on the right is used for output items that are not used in the recipe. The slot on the left is used for temerary storage and sorting. Crafted items can be pulled out of the blocks output slot using a Remote Item Teleporter block on the top of the block then linking it to another Remote Item Teleporter block.

Side Note
Front Side: Can toggle the block on or off.
Back Side: Can toggle the block on or off.
Left Side: Items are sorted and added to the crafting grid.
Right Side: Items not in the recipe will be added here.
Top Side: A Remote Item Teleporter can pull output slot items from the crafting grid.
Bottom Side: Unused

Devices

Some devices can be linked to this block to add advanced functions to the block. Padlocks can make the block protected blocking access form accessing the inventory for regular players. Breaker blocks can block interaction with toggleing the block on or off. Remote Item Teleporters can be added to the top of the block then linked to other Remote Item Teleporters to pull items from the output slot.
