Changes

Below are a list of changes made to the item.

  • NWTG Automation 0.1.0
    • Added support for checking information for machines (Auto Crafting, Auto Collecting, Auto Farming, Auto Fishing, and Auto Tree Farm).
    • Added support for checking information for devices (Remote Item Teleporter, Breaker Panel, and Padlock).
    • Added the Technology Scroll.
    • Added the Technology Scroll item texture.
    • Added the Technology Scroll forge item tag.
