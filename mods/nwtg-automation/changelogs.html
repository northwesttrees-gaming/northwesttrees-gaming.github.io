THE TACKLE UPDATE - 0.3.0

TOP TOP
  • Added a new machine block "Auto Fishing Machine".
  • Added Aluminum Armor (Helmet, Chestplate, Leggings, Boots).
  • Added Aluminum Armor crafting/smelting/blasting recipes.
  • Added Aluminum Armor is possible to unlock and craft but not smelt right now.
  • Added Aluminum blocks (Aluminum Bars, Aluminum Door, Aluminum Ore, Aluminum Trapdoor,
    Block Of Aluminum, Chiseled Aluminum, Industrial Aluminum, Polished Aluminum).
  • Added Aluminum items (Aluminum Ingot, Aluminum Nugget, Raw Aluminum).
  • Added Aluminum Tools (Axe, Hoe, Pickaxe, Shovel, Sword).
  • Added new devices (Padlock, -, -, -).
  • Added recipes for new items and blocks.
  • Added recipes for new devices and machines.
  • Added support for right-click action sounds.
  • Added tag support for all new blocks, items, armor and tools.
  • Added Technology Scroll item.
  • Added Vanilla item tags. (Comparator, Black/Yellow Dye, Fishing Rod, Honeycomb, Name Tag)
  • Changes where made to the unlocking script, now with tag support.
  • Changes to the right-click script for blocks.
  • Machine Blocks and Machine Wrenches now use aluminum ingots instead of iron ingots.
  • Tags currently have issues with custom armor but will be fixed in MCreator 2021.2, some things related to armor may be limited in the release.
  • Technology Scroll now give information about the block face when right-clicked on the machine or device.
